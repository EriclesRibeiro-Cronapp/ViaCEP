Bem vindo ao ViaCEP

ViaCEP - Sistema de busca informações por CEP e armazenar contatos cadastrados por usuário, gerando uma exibição com cards dos contatos salvos.

API

A API utilizada foi a ViaCEP. Ao fazer as buscas por CEP, ela mostra informações como logradouro, complemento, bairro, localidade, UF e outros.

Flexbox

O modelo de layout utilizado foi o Flexbox. Utilizado nos cards da página Home e Contatos. Em contatos, junto com ng-repeat vários cars com os contatos salvos são gerados e exibidos.

Consulta

Apesar das dificuldades, a consulta no banco foi gerada. Através dos blocos Servidor (Consulta dos dados) e Cliente (Exibição dos dados).
